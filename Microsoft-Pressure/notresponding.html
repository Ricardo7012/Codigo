CANT CONNECT TRY DAC: Have you attempted to connect to SQL Server using the regular connections and received a time out? If so, please select the "show tip."

Try to use the DAC connection. Open a new query window, please note that using the regular database connection opens multiple windows and wont work. Add the "ADMIN:" minus the double quotes to your connection string and connect to the instance.
